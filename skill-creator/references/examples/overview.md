# Examples Overview

This directory contains example SKILL.md files demonstrating various patterns and styles.

## Example Types

| File                                   | Description                                         |
| -------------------------------------- | --------------------------------------------------- |
| [simple.md](simple.md)                 | Basic skill with minimal structure                  |
| [with-scripts.md](with-scripts.md)     | Skill that includes executable scripts              |
| [with-resources.md](with-resources.md) | Skill that references multiple supporting documents |
| [comparison.md](comparison.md)         | Good vs bad description examples                    |

## How to Use These Examples

Read the relevant example based on your skill type:

- New to skills? Start with [simple.md](simple.md)
- Need scripts? See [with-scripts.md](with-scripts.md)
- Multiple resources? See [with-resources.md](with-resources.md)
- Writing descriptions? See [comparison.md](comparison.md)

For SKILL.md structure and content guidelines, see [references/content.md](../content.md).
